# cou
a world-changing game





### 游戏历史



2015-2016 游戏的前身出现

2017-2018 游戏在线下诞生，并且在常州一中开始流行

2019-2020 游戏的电子化方案不断地被讨论

2020.6 游戏的电子化方案获得投资，项目正式立项

2020.7.1-2020.7.9 游戏电子化方案进入策划状态，创始人开始频繁见面商议，确定开发方案

2020.7.10 游戏开发正式开工
